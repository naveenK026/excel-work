AppsFlyer Duplicate Highlighter

Sort and highlight duplicate AppsFlyer IDs.

Upload one report file and this tool will sort the data by AppsFlyer ID, then highlight every row where that ID appears more than once.

The output file is sorted by AppsFlyer ID and saved as a new highlighted workbook.

What it does

  • Finds the row that contains the AppsFlyer ID header.
  • Sorts the data below that header by AppsFlyer ID.
  • Highlights every row with a duplicate AppsFlyer ID.
  • Downloads the result as a new .xlsx file.
