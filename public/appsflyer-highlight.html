AppsFlyer Duplicate Highlighter

Sort and highlight duplicate values by any column.

Upload one report file, choose a column name, and this tool will sort the data by that column, then highlight every row where the same value appears more than once.

The output file is sorted by your selected column and saved as a new highlighted workbook.

What it does

  • Detects the header row from the uploaded file.
  • Suggests the available column names after upload.
  • Sorts the data below that header by your selected column.
  • Highlights every row with a duplicate value in that column.
  • Downloads the result as a new .xlsx file.
