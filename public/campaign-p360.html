Campaign P360 Generator

Build per-campaign P360 workbooks.

Upload the Install and InApp reports, enter the app name, and this tool will generate one Excel file per campaign ID inside a single ZIP. It supports large XLSX and CSV inputs and writes output in a streaming-friendly way.

Output files use the format AppName - CampaignNumber - p360.xlsx.

What it does

  • Reads the Campaign column from both files.
  • Extracts the last campaign number, like 1216.
  • Creates one workbook per campaign number.
  • Adds PA - Install and PA - InApp sheets when data exists.
  • If InApp is missing a campaign, the workbook includes only PA - Install.
